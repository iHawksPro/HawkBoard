# PaletteBoard Feature Backlog

## MVP

- IME service with stable alphabet/symbol/emoji layouts
- keyboard canvas renderer with theming hooks
- toolbar strip and suggestion strip
- settings dashboard with keyboard enable/select shortcuts
- theme library with preset seeding, duplicate, delete, activate
- theme builder with live preview, corner radius, spacing, label size, colors, trail color
- Room theme persistence and JSON import/export
- DataStore user settings for number row, height, glide typing, toolbar config, sound/haptics
- local glide typing decoder starter and visual trail overlay
- auto-capitalization, auto-spacing, and basic contextual suggestions

## V2

- clipboard history and pinned clipboard
- snippets/macros and long-press customization
- better multilingual switching and per-language layouts
- stronger local autocorrect and next-word predictions
- one-handed layout geometry and ergonomic resize handles
- accessibility presets including dyslexia mode and high contrast mode
- richer toolbar widgets like calculator, notes, and translation

## V3

- split keyboard for tablets/foldables
- advanced emoji/sticker panel and local search
- animation packs, RGB themes, and seasonal content packs
- advanced gesture delete, cursor swipe, and spacebar trackpad mode
- theme sharing feed or marketplace
- typing analytics dashboard and training insights

## Experimental

- AI theme generator from prompt or wallpaper
- context-aware toolbar recommendations
- custom sound pack authoring kit
- rewrite/tone tools with strict privacy gating
- adaptive haptic profiles by app or field type
